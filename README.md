# EcoPulse - Клиентская часть
## Описание проекта
Frontend-приложение для расчетов выбросов загрязняющих веществ от различных источников

## Запуск проекта
### Предварительные требования
* Node.js
* npm (обычно устанавливается вместе с Node.js)

### Через командную строку
Восстановление зависимостей
```
npm install
```
Запуск в режиме разработки
```
npm run dev
```
